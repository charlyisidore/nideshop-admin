# nideshop-admin

Node.js开源商城NideShop管理后台（基于[Node.js](https://nodejs.org/)、[Vue.js](https://vuejs.org/)、[Element Plus](https://element-plus.org/zh-CN/)）

## 测试版包含功能（待完善）

+ 管理员登录
+ 商品管理
+ 商品分类
+ 品牌管理
+ 会员管理
+ 订单管理

## Build Setup

+ 安装`nideshop-server`

[GitHub](https://github.com/charlyisidore/nideshop-server)

+ 运行

```bash
yarn install
yarn run dev
```
